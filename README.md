# Netflix Clone 🎬

A responsive **Netflix landing page clone** built using **HTML5** and **CSS3**. This project recreates the modern look and feel of Netflix's homepage, focusing on clean design, responsive layouts, and CSS styling techniques.

## ✨ Features

* Responsive design for desktop, tablet, and mobile devices
* Modern Netflix-inspired user interface
* Hero banner with call-to-action section
* Navigation bar with logo and sign-in button
* Movie and TV show content sections
* Hover effects and smooth CSS transitions
* Frequently Asked Questions (FAQ) section
* Footer with organized links
* Clean and well-structured HTML and CSS code

## 🛠️ Technologies Used

* HTML5
* CSS3
* Flexbox
* CSS Grid
* Media Queries
* CSS Animations & Transitions

## 🎯 Purpose

This project was created to practice front-end web development by replicating the design of a popular streaming platform. It demonstrates responsive web design principles, semantic HTML, modern CSS layouts, and UI development without using JavaScript or external frameworks.

> **Note:** This project is for educational purposes only. It is a static front-end clone and is not affiliated with or endorsed by Netflix. It does not include authentication, video streaming, or backend functionality.
